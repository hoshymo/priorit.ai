## 開発者向け README


## 実行のしかた

backend/.env に Gemini API key を設定する

```backend/.env
REACT_APP_GEMINI_API_KEY=xxxxxxxxxx
```

### Backend 起動

```bash
cd backend
npm i
npm start
```

### Frontend 起動

```bash
npm i
npm start
```
